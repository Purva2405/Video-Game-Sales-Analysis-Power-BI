# Video Game Sales Analysis | Power BI

## Project Overview:

This project is an **end-to-end data analytics dashboard built using Power BI Desktop**, analyzing global video game sales data to identify trends across **regions, genres, platforms, and time**.

The goal of this project is to demonstrate skills in **data cleaning, data modeling, DAX, and business-focused data visualization**.

## Dataset:

* **Source:** vgsles(CSV)
* **Data Includes:**

  * Game titles
  * Platforms
  * Genres
  * Publishers
  * Release year
  * Regional sales (North America, Europe, Japan, Rest of World)
  * Global sales

## Tools & Technologies:

* **Power BI Desktop**
* **DAX (Measures & KPIs)**
* **Data Modeling**
* **Data Cleaning (Power Query)**
* **Data Visualization & Dashboard Design**
